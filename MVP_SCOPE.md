# GameFlow MVP Scope

## In v1

- authentication
- dashboard
- backlog CRUD
- game search
- recommendations
- tonight picker
- gaming stats
- responsive web UI
- deploy to Vercel

## Not in v1

- social features
- comments
- chat
- friends system
- achievements sync
- Steam account import
- PlayStation / Xbox import
- billing
- native mobile apps
- admin backoffice
- advanced AI assistant
- notifications system

## Product rule

When in doubt, cut scope.
