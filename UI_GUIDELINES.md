# UI Guidelines

## Style

- dark-first
- modern gaming SaaS
- minimal clutter
- strong spacing
- clean cards
- responsive design

## Layout

Sidebar:

- Dashboard
- Backlog
- Discover
- Tonight
- Stats
- Settings

Topbar:

- GameFlow wordmark
- search area
- profile menu

## Key screens

### Dashboard

- stat cards
- recently added
- quick action
- tonight suggestion

### Backlog

- grid and list view
- filter by status
- sort by title, rating, date added

### Discover

- search field
- result cards
- add to backlog action

### Tonight

- small form
- 3 suggestions
- short reasons

### Stats

- 3 to 5 strong metrics
- optional single chart
- gaming DNA card

## UX rules

- every async state needs loading state
- every empty state should be useful
- destructive actions should be clear
- no crowded tables by default
