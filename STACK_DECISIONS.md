# Stack Decisions for GameFlow v1

## Chosen stack

- Next.js App Router
- TypeScript strict mode
- TailwindCSS
- shadcn/ui
- Prisma ORM
- PostgreSQL
- NextAuth or Supabase Auth
- Vercel

## Why this stack

- fast MVP delivery
- strong TypeScript support
- easy deploy
- good fit for Codex
- good path to future mobile clients via shared backend

## Hard decisions

- Web app first
- No native iOS in v1
- One external game API only
- Deterministic recommendations first, AI later
- Monolith first, not microservices
